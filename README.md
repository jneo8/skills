# Skills

My personal AI agent skills.

## Usage

```bash
just link-claude      # Link skills to Claude Code
just link-gemini      # Link skills to Gemini CLI
just link-all         # Link to both
just validate <name>  # Validate a skill
just validate-all     # Validate all skills
```

## Adding a New Skill

Create a directory with a `SKILL.md` file, then run `just link-all`.
